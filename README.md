## Currency change

In the project, you have  three differents Currencies and you are able to convert between each of them.

You will only need to:

 - Pull the master 
 - Open the command line on the root 
     - run `npm install` 
     - run `npm run dev`
 - Move you to `src/api/`
     - run `cd src/api/`
     - run `npm install` 
     - run `npm run dev`
